Welkom op Woobsite

Uitleg

Je kan h1 gebruiken voor een headtitle/beginstuk van je website

p is een element dat wordt gebruikt om teksten te maken en schrijven.

a is een element dat wordt gebruikt om hyperlinks te maken naar andere webpagina's of bronnen.

img is een element dat wordt gebruikt om afbeeldingen op een webpagina in te voegen.

table is een element dat wordt gebruikt om tabellen op een webpagina te maken.

ul staat voor ongeordende lijst en creëert een lijst zonder specifieke volgorde, meestal weergegeven met opsommingstekens.

ol staat voor geordende lijst en creëert een genummerde lijst met een specifieke volgorde.

li wordt gebruikt om individuele lijstpuntitems binnen ul of ol te definiëren.
